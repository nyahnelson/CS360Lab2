About this website

This is a personal website designed for my database systems lab. The goal of this assignment is to learn HTML and CSS, as well as integrate them and create a well-designed website. To create this website, I started by creating a new project in Netbeans IDE. After my project was made, I started creating multiple HTML files (one for every page of this website), and one style CSS file to layout the style of every HTML page.

What I did

Like I said before, I made an HTML file for every page on this website. In every HTML file, I included a link to the CSS file that has the style guidelines for each HTML page. Some things I added to the CSS file were the background color, centering certain text, coloring text, the navigation bar, and a few more details.
